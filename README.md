# 🌏 TourHub旅游輔助系統

這是一個基於 **LINE LIFF** 架構的整合型旅遊生態系統，旨在解決團體出遊的各個痛點。本專案整合了四大核心模組，涵蓋了從行程規劃、熱門探索、聚會集合到費用分擔的完整流程。

## 🚀 四大核心模組

### 1. 行程管理與置物櫃搜尋

專為自由行旅客設計的基礎管理工具。

* **行程規劃**：建立、編輯與管理旅行日程，並透過 LINE Flex Message 精美分享。
* **置物櫃搜尋**：整合 API 與地圖，支援搜尋 **台灣** 與 **日本** 地區的置物櫃，提供即時位置與價格資訊。

### 2. 智慧分帳系統

解決團體旅遊最頭痛的「誰該付錢」問題，支援前後端分離架構。

* **帳務結算**：支援多幣別匯率轉換，自動計算群組成員間的債務關係。
* **LINE 整合**：透過 LINE Login 進行身份驗證，輕鬆管理群組與帳單。
* **完整架構**：包含前端 (React)、核心後端 (Node.js) 與 Profile 服務 (Python)。

### 3. 熱門行程排行榜

透過數據驅動的視覺化看板，幫助用戶發現靈感。

* **多維度排行**：提供「即將出發」、「熱門地區」、「季節精選」等多種分類榜單。
* **社群互動**：支援收藏與分享功能，透過金/銀/銅獎杯視覺化呈現熱門程度。
* **數據統計**：即時分析瀏覽、分享與收藏數據，推薦最適合的行程。

### 4. 集合系統 (Gather)

專注於特定活動的快速集合工具（以東京澀谷為例）。

* **活動倒數**：自動計算距離活動開始的剩餘時間。
* **一鍵導航**：內建澀谷熱門地標（如 SHIBUYA SKY、忠犬八公像），結合 Google Maps 快速指引集合地點。

## 🛠 技術架構 (Tech Stack)

本平台採用現代化的微服務與無伺服器架構，確保高擴展性與維護性。

| 領域 | 技術選型 |
| --- | --- |
| **前端框架** | **React 18/19** (整合 CSS Modules 與 Lucide Icons) |
| **整合平台** | **LINE LIFF SDK**, LINE Login, LINE Flex Message |
| **後端服務** | **Node.js (Express)**, **Python (Flask / Vercel Functions)** |
| **資料庫** | **MySQL** (儲存用戶、行程、帳務與排行榜數據) |
| **地圖服務** | **Google Maps API**, OpenStreetMap Nominatim |
| **部署環境** | Vercel (支援 Serverless), 靜態託管 |

## 📦 快速開始與部署概覽

由於本平台由多個倉庫組成，請依需求分別部署各模組：

1. **環境變數設定**：所有模組皆需配置 `.env`，包含 `LIFF_ID`、資料庫連線資訊 (`DB_HOST`, `DB_USER`...) 以及 Google Maps API Key。
2. **資料庫準備**：確保 MySQL 中已建立對應的 User、Group、Bill、Trip 等資料表。
3. **安裝依賴**：
* 前端：`npm install`
* 後端 (Node): `npm install`
* 後端 (Python): `pip install -r requirements.txt`


4. **啟動服務**：
* 開發模式：使用 `npm start` 或 `npm run dev`。
* 生產部署：建議使用 **Vercel** 進行一鍵部署，前後端可自動適配。
