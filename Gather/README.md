# Tokyo Gather 🗓️

一個基於LINE LIFF的日本東京澀谷聚集活動規劃應用程式。

## 功能特點

1. **行程規劃**: 輸入日期、時間、活動名稱和地點
2. **地圖整合**: 使用Google Map搜尋和標記集合地點
3. **快速選擇**: 澀谷熱門景點預設列表（澀谷天空展望台、十字路口、忠犬八公像等）
4. **倒數計時**: 自動計算活動開始時間的倒數
5. **LINE整合**: 透過LINE LIFF支援使用者登入和分享
6. **行動友善**: 響應式設計，支援多種裝置

## 技術棧

1. **前端框架**: React
2. **地圖服務**: Google Maps API
3. **整合服務**: LINE LIFF
4. **UI圖標**: Lucide React
5. **樣式**: CSS（自訂樣式）

## 主要功能模組

### 使用者認證
1. 整合LINE LIFF進行使用者登入
2. 顯示使用者頭像和相關資訊

### 表單功能
1. **日期選擇**: 選擇聚集日期
2. **時間設定**: 設定聚集時間
3. **名稱輸入**: 輸入活動名稱
4. **地址搜尋**: 支援地址自動補全和名稱搜尋

### 地圖功能
1. 整合Google Maps API
2. 地點標記和導航
3. 澀谷預設景點快速選擇

### 分享與下載
1. 複製行程資訊
2. 分享至LINE
3. 下載活動詳情

## 使用方式

1. 啟動應用程式
2. 登入LINE帳號
3. 填寫活動資訊（日期、時間、名稱）
4. 搜尋或選擇集合地點
5. 查看地圖確認位置
6. 分享或下載行程

## 快速景點列表

應用程式內建澀谷熱門景點包括：
1. 🌟 SHIBUYA SKY（澀谷天空展望台）
2. 🚶‍♂️ 澀谷十字路口
3. 🐕 忠犬八公像
4. 及更多景點

## 依賴套件

1. `react`: UI框架
2. `@line/liff`: LINE LIFF SDK
3. `lucide-react`: 圖標庫
4. `google-maps-api`: 地圖服務（需在HTML中引入）

## 配置需求

1. LINE Channel ID（用於LIFF初始化）
2. Google Maps API Key（用於地圖功能）
3. LINE Login settings
